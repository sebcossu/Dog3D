# Dog3D
A pseudo-3D raycasting engine inspired by id Software's Wolfenstein 3D engine.  
It takes a 2D map described by a text file like the one in data folder and renders it in 3D.  
  
# Controls  
**W** = Forward  
**A** = Rotate Left  
**S** = Backward  
**D** = Rotate Right  
**J** = Move Left  
**L** = Move Right  
**Esc** = Quit
